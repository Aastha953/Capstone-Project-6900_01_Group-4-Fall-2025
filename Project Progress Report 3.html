Project Progress Report 3 – Model development Phase

In the beginning of the report, include in a table format the names of the group members who worked on this report. If a group member has not contributed to the report, leave the corresponding cell in the table blank. All group member names should appear in the table.

All groups must submit a word or pdf document addressing in writing the requirements below.  This paper is independent of size but should be concise and to the point and provides answers to the project report sections below. 
The presenting groups will provide a 10-minute presentation using a presentation app to showcase their plans for this project report. The presenting groups should upload the presentation file and the report file.

Deliverables/Requirements 
This phase is the model development phase. Include in your paper and/or presentation the following sections:

  1. Develop the three models (regression, classification, clustering, time series, or others) which you will use to make the predictions the corporations is asking for in the project and to answer the questions in the project. Each model should be developed using the Azure AutoML platform or other AutoML Platform. For each model, you need to assess and define your independent variables, your dependent variable(s) and write and explain the importance of each variable for the task or tasks of the project. Refer and write about all the hyperparameters you have set and the hyperparameter values you have used.

  2. Once the models have been developed using AutoML, develop all three of them using Python and using the algorithms and hyperparameter settings you have used in your AutoML models. For each model you have developed in Python, include in your paper as an appendix the code you have used.
     
  3. Justify in detail the technical and business reasons for which you have selected and used the algorithms and models for your work. Research the industry and make references to the corporate world as to how these algorithms and models are used in the industry, their diffusion in the market, industry segments they are used, and tasks they are used for such as sentiment analysis, fraud detection etc.

Make sure your work is pragmatic and not theoretical and that the models and tools you propose will really help the company with the questions in the project.
